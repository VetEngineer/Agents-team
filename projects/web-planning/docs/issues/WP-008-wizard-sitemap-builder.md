# WP-008 Wizard step: sitemap + section builder

## Priority
P0

## Summary
Build the sitemap and section builder with presets and drag ordering.

## Scope
- Page list with add, duplicate, and reorder actions.
- Section list with presets and drag ordering.
- Section editor panel (side sheet on desktop, bottom sheet on mobile).

## Acceptance Criteria
- Users can add/reorder pages and sections.
- Section presets populate default fields.
- Ordering is persisted via autosave.

## Dependencies
- WP-005 Core CRUD APIs + autosave
