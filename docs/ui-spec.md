# UI Spec (Survey + Web Builder)

## Global
- Layout: Navigation Rail (desktop) / Bottom Bar (mobile)
- App Bar: Project name + autosave status + exit
- Feedback: Snackbar for saved/error
- Input: Outlined text fields, helper text

## Screen: Project List
- Card grid with status chip (draft/submitted)
- Primary FAB: Create new project
- Card actions: open, duplicate, delete

## Screen: Survey Flow
- Left: step list + progress
- Right: step form
- Each question in a Card with title/description
- Example toggle on long prompts

## Screen: Page Builder
- Page list panel with + add page
- Page detail panel with + add section
- Section card shows type, title, CTA
- Drag handle for ordering
- Section editor in side sheet (desktop) / bottom sheet (mobile)

## Screen: Style Guide
- Chips for mood keywords
- Color input list with HEX validation
- Font preference select
- Photo style select (real/studio/model)

## Screen: Features/Responsive
- Checkbox list for required features
- Toggle for mobile-first
- Textarea for responsive notes

## Screen: References
- URL + reason inputs
- Avoid toggle

## Screen: File Upload
- Upload card per requirement
- Spec toggle (format, min size, usage)
- Status chip: pass/fail
- Inline guidance on failure

## Screen: Admin - Question Templates
- Left: section list with counts
- Right: question list (drag sortable)
- Edit panel: title/description/type/required
- Industry tags, show-if conditions

## Screen: Admin - Upload Requirements
- List of asset types
- Editor: formats, min size, required, helper text
- Preview panel
